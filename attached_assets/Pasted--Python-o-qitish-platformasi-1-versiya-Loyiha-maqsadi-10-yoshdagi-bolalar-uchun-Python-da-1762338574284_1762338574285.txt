## API endpointlar
- GET /api/lessons/daily - Kunlik darslar
- GET /api/lessons/{id} - Bitta dars
- POST /api/lessons/generate - Yangi dars yaratish
- GET /api/stats - Statistika

## Telegram bot komandalari
- /start - Botni ishga tushirish
- /daily - Kunlik darslar
- /lesson {id} - Ma'lum bir dars
- /help - Yordam

## O'rnatish va ishga tushirish
### Neon PostgreSQL sozlash
1. [Neon Console](https://console.neon.tech) ga kirib yangi project yarating
2. Database connection string ni oling
3. .env fayliga qo'shing

### Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend:
cd frontend
npm install
npm start

### Telegram bot:
cd backend
python bot/telegram_bot.py

## Kerakli API kalitlari
- GEMINI_API_KEY - Google Gemini API
- TELEGRAM_BOT_TOKEN - Telegram bot token
- TELEGRAM_CHANNEL_ID - Telegram kanal ID
- DATABASE_URL - Neon PostgreSQL connection string
